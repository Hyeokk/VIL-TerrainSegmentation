# Dataset Guide

This project uses three off-road semantic segmentation datasets. Each must be downloaded separately and placed under the `data/` directory. Only RELLIS-3D is required; RUGD and GOOSE are optional and will be auto-detected if present.

```
data/
├── Rellis-3D/
├── RUGD/
└── GOOSE/
```

---

## RELLIS-3D

Off-road multi-modal dataset collected on the Rellis Campus of Texas A&M University. Contains 6,235 annotated images across 5 traversal sequences on non-paved trails with bushes, sparse trees, pastures, and forested areas.

- **Source**: [https://github.com/unmannedlab/RELLIS-3D](https://github.com/unmannedlab/RELLIS-3D)
- **License**: [CC BY-NC-SA 3.0](https://creativecommons.org/licenses/by-nc-sa/3.0/)
- **Resolution**: 1200 x 1920 (H x W)
- **Original classes**: 20
- **Used in this project**: RGB images + ID-coded segmentation masks (LiDAR, stereo, ROS bags are not used)

### Download

| File | Size | Link |
|------|------|------|
| Full Images | 11 GB | [Google Drive](https://drive.google.com/file/d/1F3Leu0H_m6aPVpZITragfreO_SGtL2yV/view?usp=sharing) |
| Image Annotations (ID format) | 94 MB | [Google Drive](https://drive.google.com/file/d/16URBUQn_VOGvUqfms-0I8HHKMtjPHsu5/view?usp=sharing) |
| Image Annotations (Color format) | 119 MB | [Google Drive](https://drive.google.com/file/d/1HJl8Fi5nAjOr41DPUFmkeKWtDXhCZDke/view?usp=sharing) |

For training, download **Full Images**, **Image Annotations (ID format)**, and **Image Split File**.

### Directory Structure

```
data/Rellis-3D/
├── 00000/ ... 00004/                      # Sequence directories
│   ├── pylon_camera_node/                 # Raw RGB images (.jpg)
│   ├── pylon_camera_node_label_color/     # Color-coded segmentation masks
│   └── pylon_camera_node_label_id/        # ID-coded segmentation masks (used for training)
├── split/                                 # Official split files
│   ├── train.lst
│   ├── val.lst
│   └── test.lst
└── split_custom/                          # Generated by: python scripts/make_split_custom.py
    ├── train_70.lst                       # 70% training set
    └── test_30.lst                        # 30% test set
```

## RUGD

Robot Unstructured Ground Driving dataset captured from a small unmanned ground robot traversing parks, trails, forest paths, and semi-urban areas. Contains 18 video sequences with dense pixel-wise annotations for every fifth frame.

- **Source**: [http://rugd.vision/](http://rugd.vision/)
- **License**: [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)
- **Resolution**: 550 x 688 (H x W)
- **Original classes**: 24
- **Used in this project**: RGB frames + color-encoded annotation PNGs

### Download

Download both the annotated frames and annotation files from the official website: [http://rugd.vision/](http://rugd.vision/)

The website provides per-sequence ZIP files for frames and annotations. Download all sequences and extract them into the directory structure shown below.

### Directory Structure

```
data/RUGD/
├── RUGD_frames-with-annotations/          # RGB frames
│   ├── creek/
│   │   ├── creek_00000.png
│   │   ├── creek_00005.png
│   │   └── ...
│   ├── park-1/
│   ├── park-2/
│   ├── park-8/
│   ├── trail/
│   ├── trail-1/ ... trail-15/
│   ├── village/
│   └── forest/
└── RUGD_annotations/                      # Color-encoded segmentation masks
    ├── creek/
    │   ├── creek_00000.png
    │   └── ...
    ├── park-1/
    └── ...
```

Each annotation pixel's RGB color maps to one of 24 classes. The color-to-class mapping and conversion to the unified 7-class scheme is handled by `src/dataset.py`.

## GOOSE

German Outdoor and Offroad Dataset (GOOSE) for perception in unstructured environments. Contains annotated RGB and NIR images with 64 semantic classes, collected from various European outdoor and forest scenarios.

- **Source**: [https://goose-dataset.de/](https://goose-dataset.de/)
- **GitHub**: [https://github.com/FraunhoferIOSB/goose_dataset](https://github.com/FraunhoferIOSB/goose_dataset)
- **License**: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
- **Resolution**: Various
- **Original classes**: 64
- **Used in this project**: Visible RGB images (`*_windshield_vis.png`) + semantic ID labels (`*_labelids.png`). NIR images and point clouds are not used.

### Download

Download the 2D training set ZIP from the official website: [https://goose-dataset.de/](https://goose-dataset.de/)

Navigate to "Download Dataset" and download the 2D image segmentation training data. Alternatively, use the download scripts provided in the [official GitHub repository](https://github.com/FraunhoferIOSB/goose_dataset).

### Directory Structure

```
data/GOOSE/
└── train/
    ├── images/
    │   └── train/
    │       ├── 2022-07-22_touareg_flight/
    │       │   ├── ..._windshield_vis.png     # Visible RGB image (used)
    │       │   ├── ..._windshield_nir.png     # Near-infrared image (not used)
    │       │   └── ...
    │       ├── 2022-09-22_touareg_neubiberg/
    │       └── ...
    ├── labels/
    │   └── train/
    │       ├── 2022-07-22_touareg_flight/
    │       │   ├── ..._labelids.png           # Semantic ID label
    │       │   └── ...
    │       └── ...
    └── goose_label_mapping.csv                # Class ID to class name mapping
```

The `goose_label_mapping.csv` file defines the mapping from integer label IDs to class names. The code reads this CSV and maps each of the 64 GOOSE classes to the unified 7-class scheme.
